This repo (`presonal-website-v2.0`) is Nurjol Badyelkhan's personal CV/portfolio site. React 18 + Vite + Tailwind + Framer Motion, single-page with sections Hero/About/Skills/Experience/Portfolio/Contact. Entry `src/App.jsx` → `src/components/`.

Nurjol: AI Engineer @ E-Mongolia (Mar 2025–present), prev Data Scientist @ Novelsoft LLC. BSc ICE from Inha University Korea (full ILWOO scholarship). GCP ACE certified. Built the MOE Analytics Dashboard for Mongolia's Ministry of Education.

**Why:** The site is his job-search showcase — quality and polish matter a lot to him.
**How to apply:** Treat design/UX quality as high-stakes. The chosen aesthetic is "Pixel AI Hacker": dark bg #050816, neon trio cyan #00f5ff / magenta #ff0080 / lime #00ff88, glassmorphism, pixel grid, Three.js particle background, custom neon cursor, and pixel-art avatar mascots that swap per section.
